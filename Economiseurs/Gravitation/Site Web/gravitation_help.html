How to create and use your own graphics ?

With Gravitation, you can use your own pictures via a 'skin' file. To create your own skin, you can start from the two provided skins that are installed with Gravitation. I will assume that you know how to create/draw a bitmap and how to manipulate bitmap files.

What software to use:

  • To create and manipulate the pictures, use any graphic editor or 3D program, or ray-tracer you want ( Photoshop, PaintShop Pro, POV... ).
  • To paste each individual animation frame into one file for one planet, use the skinmaker utility you can download from this site.
  • To edit the skin description file, use any text editor you are used to.

To create a skin, you muste create:

  • A bitmap for each planet. This bitmap file must be a 256 colors DIB format file. It is a horizontal band containing each individual frame of the planet animation. All planets must share the same 256 colors planet.
  • A 256 colors palette file that will contain all colors used by the planets. One color must be reserved for the transparent areas.
  • A skin description text file. I suggest you start with the provided samples.

Creating the pictures:

  • Draw all your planet frames. Group the frames of each planet into a single horizontal band. The SkinMaker utility can help you doing that. At this time, you can create the bitmap in any color depht you want, but consider that all pictures will share the same 256 colors.
  • Save all your planet bands into a single file.

Sharing the 256 colors among the planets:

  • Paste all the pictures into a single bitmap
  • Decrease this bitmap color depth to 256. You will get the global color palette that will be shared by all your skin's pictures.
  • Decrease each planet picture color depth to 256 using the global palette. Save them as DIB files.
  • Create a new palette.dib from the sample. Change the color indexes to your global palette.

Creating the skin description file:

  • Start from one of the sample skin description file that is installed with Gravitation. Be sure you work on a copy of the file.
  • Open the description file with your favorite text editor program.
  • Edit it to reference your own graphic file and colors.
