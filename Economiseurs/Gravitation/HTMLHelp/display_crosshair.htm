Display crosshair

This options lets the program display a crosshair on the heaviest planet or star.

This can be useful to know what is the 'center' of the star system.

WARNING: this option can cause your computer to hang if set. This only happens with some computer, it may work with yours.   If your computer hangs, you will have to restart it. This is a known bug I could not fix yet. Please disable this option if it hangs your computer. Please close all your applications and save all your data before trying this option.

I hope I can soon fix this bug, watch my web page for new versions.
