Economiseur d'écran Gravitation
 
 
Cet économiseur d'écran simule la gravitation universelle s'appliquant entre des corps célestes. Devant vos yeux, se créent et se détruisent des planètes et des systèmes solaires. Les corps célestes entrent en collision, explosant en petits morceaux, ou créant un nouveau corps plus gros qui se transformera peut être en étoile.

Vous n'avez pas moins de 6 pages de paramètres pour ajuster votre univers à vos désirs.

Caractéristiques techniques

Gravitation utilise des formules mathématiques réelles pour simuler la force de gravité entre tous les corps célestes. Vous avez la possibilité d'utiliser un modèle physique à
    2 deux dimensions. Ce modèle est plus rapide, mais moins réaliste.
    3 dimensions. Ce modèle est plus réaliste, mais moins rapide.

Configuration

    La configuration de ce programme est extrèmenent souple, vous disposez d'une foule de paramètres.
    Pour accéder à la fenêtre de configuration, ouvrez la fenêtre de Propriétés du Bureau en cliquant sur le bureau de Windows avec le bouton droit de la souris. Choisissez l'onglet des économiseurs d'écran. Sélectionnez "Gravitation" dans la liste de vos économiseurs d'écran. Vous obtenez immédiatement un aperçu de l'aspect de l'économiseur.
    Cliquez sur le bouton "Paramètres" pour ouvrir la fenêtre de configuration de Gravitation.
    La fenêtre de configuration de Gravitation contient six onglets représentant six pages de paramètres.

    Un conseil, modifier chaque paramètre prudemment, un par un, certains peuvent avoir une très grand influence sur le comportement du programme et vous amener des systèmes solaires complètement instables.

Onglet "Informations"

    Cette page vous donne quelques informations générales sur le programme.
 

Onglet "Nombres"

    Cette page vous laisse le choix du nombre de planètes que le programme gèrera.
    Nombre minimum:    Quand les collisions successives ne laissent plus suffisament de corps, un nouveau système est généré.
    Nombre initial:    Nombre de corps céleste créés pour chaque nouveau système.
    Nombre maximal:    Le nombre maximum de corps que le programme gèrera. Des collisions peuvent créer de nouveaux corps.
    ATTENTION ! Le calcul de gravité augmente avec le carré du nombre de corps.

Onglet "Création"

    Cette page vous permet de choisir la position et la vitesse initiale des corps célestes. Le programme est capable de placer les corps sur des orbites circulaires autour d'autre corps.

    Vous pouvez également choisir entre le modèle physique 2D et 3D. Sachez cependant que l'affichage ne se fera en aucun cas en trois dimensions. J'ai voulu garder un affichage le plus fluide possible. Cependant, en 3 dimensions, les corps les plus éloignés passeront derrière les corps les plus proches ( appelont ça 2D 1/2 )..

Onglet "Physique"

    Cette page vous permet de régler le modèle physique utilisé par le programme. ATTENTION ! Certains paramètres sont difficiles à régler et le programme ne peut pas faire tous les contrôles. Certaines valeurs peuvent provoquer des plantages, allez y prudemment. Essayez de ne modifier qu'un paramètre à la fois.
     
    Constante de gravité (G) Ce paramètre permet de régler l'intensité de la force de gravitation qui s'exerce entre les corps.
    Mult. de temps  Permet d'accélérer ou de ralentir le temps pour une animation plus rapide.
    Mult. de distance  Permet de régler la relation entre les distance à l'écran ( en pixels ) et les distances dans le modèle physique. Sachez que la force de gravité décroit avec le carré de la distance. La taille visible des corps célestes ne varie pas.
    Mult. de rayon :  Ce paramètre est utilisé pour les calculs de collision. Diminuez le pour permettre aux corps de s'approcher plus près les uns des autres.
    Pourcent de vitesse de libération : Quand un corps céleste s'éloigne trop du corps central, le programme calcule sa vitesse de libération (pondérée par ce pourcentage ) pour déterminer si ce corps doit être considéré comme perdu dans l'espace. Un coefficient trop proche de 100 aura pour consequence que des corps éloignés reviendront après une TRES longue période de temps.
    Vitesse de collision max Quand deux corps de masse proche entrent en collision, l'effet produit dépend de leur vitesse de rencontre. Si celle-ci est suffisament faible, les deux corps s'agglomèrent pour former un corps plus lourd. Sinon, ils explosent en une multitude de fragments plus petits.
     

Onglet "Comètes"

    Cette page permet de régler les paramètres des comètes. Les comètes sont des corps célestes pourvu d'une chevelure formée par les gaz et poussières qui s'en échappent que elle s'approche d'une étoile. Cette chevelure est repoussée par le vent solaire. Ces paramètres ont surtout une influence sur l'affichage.
    • Vent solaire :    Intensité du vent solaire, détermine la longueur de la chevelure.
    • 1/fréquence de création:    Quand le nombre de corps devient relativement faible, le programme génère aléatoirement des comètes. Plus ce paramètre est grand, plus la fréquence de création de comètes est faible.
    • Nb. Max de poussières : nombre maximum de grains de poussières gérés simultanément par le programme.
    • Taille max des grains :    taille maximum de chaque grain de poussière ( en pixels ).
    • Durée de vie des grains : nombre de secondes qu'un grain de poussière reste visible à l'écran
    • % de dispersion des poussières : pourcentage MOYEN de dispersion des grains de poussière à partir d'une comète.

Onglet "Masses"

    Cette page permet de régler la masse minimale de chaque type de corps. Quand un corps change de masse ( après une collision ); sa nouvelle représentation est choisie en fonction des masses saisies dans cette page.
    ATTENTION ! Il convient généralement de régler la constante de gravité quand on modifie les masses pour garder une animation intéressante.

Onglet "Affichage"

    Cette page vous permet de régler les propriétés d'affichage du programme. Les curseurs vous permettent de choisir dans les plages de valeurs possibles ( les valeurs les plus hautes sont obtenues en dirigeant le curseur vers la droite ).
     
    Images par seconde Nombre de fois par secondes ou le calcul du modèle physique est interrompu pour afficher une image. Cette valeur doit rester autour de 25 images par seconde pour garder une fluidité agréable.
    Vitesse de travelling Vitesse à laquelle la caméra se déplace pour suivre l'astre le plus lourd du système.
    Facteur de zoom max Grossissement maximum. Les images des planètes gardent la même taille, c'est la relation entre les distances physiques et les distances à l'écran qui sont modifiées. Ce paramètre fonctionne avec le multiplicateur de distance.
    Densité du champ d'étoiles Densité d'étoiles affichées en arrière plan.
    Afficher le repère Permet d'afficher un cadrillage centré sur l'astre le plus lourd. Ce cadrillage suis aussi le zoom.
    Effacer les trajectoires Cette option permet d'effacer ou non les trajectoires des planètes. Garder les trajectoires peut être utile pour mieux visualiser les orbites.
    Changer de résolution d'écran Cette option permet au programme d'essayer une meilleure résolution d'écran. Certaines cartes vidéos provoque un arrêt immédiat de léconomiseur d'écran quand on modifie la résolution. Si tel est le cas pour votre machine, vous devez décocher cette option.
