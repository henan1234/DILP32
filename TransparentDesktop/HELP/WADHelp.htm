Principle

The principle is very simple: You just add a transparent layer on you screen ( in front or behind your windows ).
You simply draw what you want on the layer using your favorite image editor ( if you don't have one, just use the standard Windows Paint program ).

You can have as many layers ( or pages ) as you want, depending only on how many disk space you want to use.
You can protect your notes with a password, even can you have a different password for each page.

Quick start

  • Launch "TransparentDesktop.exe"
  • It is a good idea to add a shortcut to "TransparentDesktop.exe" to your Startup directory if it has not be made by the install program, so Transparent Desktop will be automatically launched every time you start Windows. If you installed the program with the setup program, the shortcut has been automatically created for you.
  • A small icon appears on the tray bar
  • Right click on the icon to display the menu and access the configuration window
  • Left click on the icon to move the note to foreground
  • Double clic on the icon to edit the notes.

    • Simple, isn't it ?


    The tray bar icon
    When you start Transparent Desktop, a small icon is displayed in the task bar notification tray.
    Left clic on the icon to move the notes to the top of the windows.
    Left double clic on the icon to edit the notes with your favorite image editor.
    Right clic on the icon to open the menu.
    Different look of the icon :

     The Menu
    The menu is displayed when you right clic on the tray bar icon or on a note. The following commands are available :
    "Display"    This lets you choose what page you want to display (you can have multiple pages only with the registered version ). The currently displayed page is checked.
    "Configure..."    Displays the configuration window.
    "Edit background"    Lets you edit your notes page.
    "About..."    Displays informations about the software
    "Help..."    Displays this file
    "Exit"    Exit from the program

    Editing your notes

    The first thing you need to know is that you will edit your notes with your favorite image editor. You can change this editor in the Configuration Window.
    You can draw whatever you want on the page.
    Just follow one rule: Whatever you will draw in black will be the transparent part of the layer.
    Don't forget to save your work before exiting the image editor. You must save the file without changing its name or format ( use "Save", not "Save As" ). You can anyway change the color depth of your image, read the 256 colors mode problem.
    When you exit from the image editor, Transparent Desktop will scan your new notes.
     

    The configuration Window

    The configuration window can be display by choosing the "Configure" command accessible form the tray bar icon menu.
    Press the "Help" button to display this file.
    Press the "Hide" button to hide the configuration window. 

    Graphic editor window

    This window lets you choose your favorite image editor. You can use Ms Paint by default, but you can use any bitmap editor you want.
    Clic the "Change..." button to choose your favorite image editor.
    Clic the "Default" button to reset Microsoft Paint as your image editor. 

    Pages window

     This window lets you manage your notes pages. You can add new pages, delete pages, protect pages with a password. 

    Managing pages

    Press the "Add new page" to add a new page
    Press the "Delete page" to delete a page 

    Password protection

    Each page can be individually password protected. You will be asked for the password each time the page will be opened ( except if you already typed the password in one session ).
    Check the "Password protected" to protect the page with a password. The default password is empty, you can change it by pressing the "Change password" button. 

    Optimizing your notes

    YES, you can use any note shape you want ( i promised it ) !
    BUT ! Some shapes are better than others. If you want Transparent Desktop Window to use less memory and Windows to run quicker, you can use optimizable shapes. This shapes should be made of rectangles :
     
    Unoptimizable shapes work fine, but they cause Transparent Desktop to generate many rectangles, so it occupies more memory space and can slow down Windows. Experiment, if you have a fast PC with plenty of memory, you may not notice any difference. 

    The 256 colors mode problem

    Microsoft never make programmer's work easy. Sometimes, even not easy for the end user. The 256 colors mode is a very weird mode, because, you don't really 256 system colors. You just have 16 like the 16 color mode. Only the foreground application can ask for 256 colors. And the Transparent Desktop is made to work in background, so if it use its 256 colors, how will your foreground text processing application look ? So, you will have to try. If your screen displays 256 colors, you can try to translate your notes page to 16 colors. 

    The 64K colors problem

    If your screen displays 64k colors, you may encounter problems when editing the page generated by Transparent Desktop with an application like Paint Shop Pro. This is a known bug. The solution is simple : when you add a new page, edit it with Microsoft Paint for the first time, save it. You will now be able to edit it with any image editor.
